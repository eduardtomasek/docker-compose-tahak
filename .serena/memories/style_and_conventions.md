UI/style conventions: dark GitHub-like theme defined inline in the `<style>` block inside `docker-compose-tahak.html`. CSS uses `:root` custom properties for colors and fonts, `flex` for header/nav, `grid` for content cards, and a single responsive breakpoint at `max-width: 700px`. Prefer reusing existing classes like `.grid`, `.card`, `.entry`, `.tip`, `.badge`, and update tokens in `:root` instead of introducing ad-hoc colors.
